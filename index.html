SUMMARY: LATITUDE VS. X

The purpose of this project was to analyze how the weather changes as you get closer to the equator. To accomplish this analysis, the OpenWeatherMap API was utilizied to assemble the dataset of over 500 cities across the world and to create a representative model.


After assembling the dataset, Matplotlib was used to plot various aspects of the weather vs. latitude. Factors analyzed includes: temperature, cloudiness, wind speed, and humidity. This site provides the souce data and visualizations created as a part of the analysis. Explanations and descriptions of any trends and correlations witnesseed are provided.
